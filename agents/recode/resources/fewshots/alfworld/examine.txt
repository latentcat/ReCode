我会像对待任何物品一样搜索台灯：导航、需要时检查、关闭时打开，当我看到它时，提取其 ID 并使用它。紧接着，我会检查 `obj_ID`，以便"看"发生在灯下。常见错误：忘记最后的 `examine {obj_ID}`，或不提取完整的 `look_obj_ID`。


<execute>
for location_ID in all_location_IDs:
    obs = run(f'go to {location_ID}')
    if 'Nothing happens.' in obs:
        obs = run(f'examine {location_ID}')
    location_closed = 'closed' in obs
    if location_closed:
        obs = run(f'open {location_ID}')
    if look_obj in obs:
